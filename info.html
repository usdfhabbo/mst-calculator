Instructions

Before using this calculator to generate your report, it's important you Check Some Things and Understand How to Use the calculator.

Check Some Things

System Time

It's important to know that this calculator uses your systems time as a reference when checking MST. For this reason, you need to ensure that your system time is correct. You can check your system time by looking in the bottom right hand corner (if you're on a PC) or in the upper right hand corner (if on a Mac). Ensure both the Date and Time are correct by comparing it to the time on your smartphone or by comparing it to an online time server, like this one.

Current MST Values

The next thing you need to check before generating your report is to ensure that the MST listing on the calculator are the same as the MST listing on Regulation 5. Here is a quick link to Regulation 5 - Promotional Procedure and Requirements.

The MST times that the calculator currently uses are on the table below. Compare this table to the one in Regulation 5.

Browser Requirements

In order for this calculator to work, your browser must have JavaScript enabled. This application was originally only tested on Google Chrome but should work with the newest version of any modern browser. Like anything else on the internet, IE is not recommended.

How to Use

Using the calculator is extremely simple and will greatly increase the efficiency and accuracy of your MST report.

There are essentialy 3 different parts to the Calculator. Those being: Input Soldier Info, Preview and BBCode. Each are explained in detail below.

Input Soldier Info

This section is where you will copy and paste the soldier's information from the PTS. Within this section, you'll find several text boxes and select menus which you will fill out for each individual soldier. The individual fields are:

  • Name
  • Join Date
  • Last Online
  • Rank
  • Duty Type

It is very important that you copy and paste both the Name and Join Date from the PTS and do not type them in manually. This is to ensure that the name is spelled correctly. Also, the calculator can only properly calculate using the same date format as the PTS, so typing in something else may produce unexpected and inaccurate results.

Last Online must be pulled from within the Habbo client by searching each soldier's name individually.

Rank and Duty Type are both found on the PTS profile of each soldier. If you cannot find a listed duty type, that soldier is Active Duty. The calculator automatically defaults to active duty.

Once you've entered an individual soldier's info, the next two sections will automatically update themselves once you press the DO IT button.

Preview

The preview section simply shows the table contents that will be placed into your PM. It's important to double check this as you're entering soldiers into your report. If you make any errors, write them down and correct them in your PM later.
It's important to note that the header and other information is not included in the preview, but all code is provided for the BBCode section for you to copy and paste into your PM

BBCode

This is where the final, generated code will be placed. Just copy and paste all of the BBCode into a new PM (using the BBCode view) and then change to Preview mode and then update the information in the header with whatever applies to your report.
